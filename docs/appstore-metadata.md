### App Review Information（外部テスター申請時 / Apple審査チーム向け）

```
マイク: 騒音レベル（音量値）の測定のみ。録音・保存・送信なし。
HealthKit: HRV読み取りのみ。書き込みはユーザーがオプトインした場合のみ。
カレンダー: 次のイベントの開始時刻のみ参照。タイトル・詳細・参加者は取得しない。
ログイン: 不要。起動後すぐにオンボーディングを開始できる。
バックグラウンドオーディオ: UIBackgroundModes:audio 設定済み。セッション中に音が継続するのは意図した動作。

Microphone: Used only for ambient noise level measurement (volume). No audio is recorded, stored, or transmitted.
HealthKit: Read access for HRV only. Write access is optional, used only when user opts in to log sessions.
Calendar: Only the start time of the next event is accessed. Title, details, and attendees are never read.
Login: Not required. Onboarding starts immediately on launch.
Background audio: UIBackgroundModes: audio is configured. Continuous audio during a session is intended behavior.
```
